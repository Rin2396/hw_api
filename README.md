## Скопируйте репозиторий
```bash
git clone https://github.com/Rin2396/hw_api
```

## Перейдите в папку с репозиторием
```bash
cd hw_api
```

## Скачайте requirements.txt
```bash
pip install -r requirements.txt
```

## Перейдите в папку reciepts_app
```bash
cd reciepts_app
```

## Запуск
```bash
flask run
```

## Запуск по порту
```bash
flask run --port=5001 
```
